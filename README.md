## ğŸ“± Application Workflow

1. The app sends an HTTP request to the Fake Store API.
2. The JSON response is parsed into a Product model.
3. Products are displayed in a GridView.
4. Users can:
   - Search products
   - Filter by category
   - View product details
   - Add items to cart
   - Switch between light and dark mode
5. Cart state is managed using Provider.
